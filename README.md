# ejercicios-vuejs
Ejercicios de practica sobre el libro **La elegancia de Vue.js 2**

### Ejecutar en local
1. Instalar las dependendias con:
```
npm install
````

2. Ejecutar el siguiente comando para levantar un servidor local y entrar a la direccion http://localhost:8080

```
http-server
```

Ejemplo:

```
$ http-server
Starting up http-server, serving ./
Available on:
  http://10.0.0.28:8080
  http://127.0.0.1:8080
Hit CTRL-C to stop the server
http-server stopped.
```
