Las directivas siempre actuan sobre un elemento o bloque de elementos html

La directiva v-show, muestra u oculta ciertos elementos dependiendo de si la variable o propiedad de $data esta presente. Con v-show los elementos siempre se renderizan ya que solo les aplica un display de CSS.

El siguiente ejemplo, muestra el uso de v-show. Asi que cuando el textarea tenga un mensaje se mostrara un boton, pero cuando este vacio. mostrara una advertencia de que tenemos que escribir un mensaje.
